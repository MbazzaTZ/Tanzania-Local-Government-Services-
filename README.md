# 🇹🇿 Tanzania Local Government Portal

> **Empowering Digital Governance for All Citizens**  
> A next-generation platform built to bridge Tanzanian citizens, local governments, and service providers through digital applications, verification, and communication.

---

## 📘 Overview

The **Tanzania Local Government Portal (TLGP)** allows:
- Citizens (*Wananchi*) to apply for services, report issues, and access official documents.
- Administrators (*Officials*) to verify, approve, and manage applications.
- Real-time engagement between government and the people.

---

## 🏗️ Tech Stack
- React + Vite
- TailwindCSS + Shadcn UI
- Supabase (Auth + Database)
- Firebase Firestore (Realtime Notifications)
- Context API + Hooks
- Node.js + npm
